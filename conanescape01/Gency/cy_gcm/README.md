
# Gency/sub_module_for_cybird/cy_gcm
Cybird製アプリ向け。  
gcm_sub_toPUSHを利用してCybird 社内PUSHサーバーからアプリにNotificationを送信する機能を提供する。  
また、社内向けID付与ルールを実装している。

 2.2.7


## Requires:
 minSdkVersion 14

 com.gency:cy_gencydid:*
